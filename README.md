## 📌 Project Overview

**FoodFlow Analytics** is a data-driven exploration of an **Online Food Delivery Dataset** designed to uncover meaningful customer and business insights.

This project analyzes customer demographics, ordering behavior, income patterns, occupations, feedback, and location trends to understand what drives online food delivery decisions.
